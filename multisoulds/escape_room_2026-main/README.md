# Escape room control panel and player interface


## Running the app locally

### Installing the dependencies
Run the following command to install the dependencies (after installing python).

```pip install -r requirements.txt```

### Running the serevr
```python main.py```

The page should now be available at [http://localhost:8000]()

The admin page is available at [http://localhost:8000/admin]()



## Building in docker
Run the following command to build the container:
```docker build -t my-escape-room .```
